◆ InfixtoPrefix()

void InfixtoPrefix ( char *  exp)

Function that rewrites an Infix Expression into Prefix form.

In this function, we call the getPostfix(char *) function between 2 reversal functions.

The conversion to prefix has been achieved by first getting the postfix of the reverse of the input expression, and then reversing the output again.

Note
We convert Infix Expression to Postfix form to Prefix form because it yields a more readable code than a function that directly converts Infix Expression to Prefix form.

◆ precedence()

int precedence ( char  c)

This function returns the precedence of any input character.

The assigned precedence is 4 for '~', 3 for '*', 2 for '+' and 1 for '>'. The function returns 0 if the input character is an operand(alphabet) or a parenthesis.

Returns
Precedence of any input character
Warning
Illegal symbols in the input expression will terminate the program. In that case , we directly exit the program after alerting the user what the error causing character is
