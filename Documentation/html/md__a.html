Foreword

We are grateful for getting the opportunity to work on this Logic in Computer Science assignment together.

Our first idea on seeing the assignment was the use of recursion, the most standard way to traverse and perform general operations on trees.

There was some thought of alternate iterative methods but observed that they were taking up around double the time, with much less readability as well.

We have thought of using C considering it is one of the closest languages to the machine. We thought so keeping the time of running in mind.

To handle the memory, we decided to make it as efficient as possible using techniques of Dynamic Memory Allocation, like malloc.

We used a lot of pointers, to minimise the length of the code, and by avoiding writing more and more functions, we tried to make the code as readable as possible.

We've also made sure every meaningful snippet of code that has a scope to be used again has been made into a function to achieve reusability of code.

Each task has its own complexity, therefore, the main focus was to achieve the simplest and most understandable solution as possible, and to make it an easy reading experience for all.

Thank You.
